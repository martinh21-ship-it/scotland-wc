Score Admin · Scotland Tracker

Enter finished scores, tap Generate, copy the result, and paste it into results.json on GitHub.

How to use: 1) Type scores for finished games (leave others blank). 2) For a game in progress, tap LIVE and enter the current score. 3) Tap Generate file at the bottom. 4) Tap Copy. 5) On GitHub, open results.json → edit (pencil) → select all → paste → commit. Friends see the update within a minute.
